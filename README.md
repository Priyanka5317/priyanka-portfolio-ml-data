# Priyanka Mangrulkar | AI, Data Science, Data Engineering Portfolio

A role-targeted portfolio for AI Engineering, Data Science, and Data Engineering.
Single-file static site (HTML + Tailwind via CDN + Three.js) with a GPU particle
nebula background and a cinematic intro. No build step.

## Deploy on Vercel
1. Import this repo at https://vercel.com/new
2. Framework preset: Other
3. Deploy.

## Structure
- `index.html` : the entire site
- `images/` : profile photo and project graphics
